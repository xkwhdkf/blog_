# 캘린더

## 목표
일정 수정, 추가, 삭제 가 가능한 스케줄러를 만들어 보자.

## 스택
React.

Tailwind.

redux.

firebase
## 공부
Tailwind.

redux.

firebase.
## 기능구현.
 - 월 이동 가능.
 - 일정 추가.
 - 일정 수정.
 - 일정 삭제.
 - 일정 선택시 모달창 상세 내용 보이기.
 - 캘린더 내에 시간순으로 일정 정렬.
